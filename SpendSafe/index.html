    About SafeSpend

    A high-fidelity, secure, client-side personal ledger built specifically for students to monitor educational overhead, log daily expenses, and track personal budgets.

    Application Purpose

    SafeSpend was engineered as a professional, vanilla web development portfolio showcase. It demonstrates modular client-side state architectures, secure manual currency conversion pipelines, complex client-side validation loops via regular expressions, and semantic web designs engineered from a strict mobile-first paradigm.

    Architecture Commitments

    • Zero Framework Footprint: Built completely with native Web Technologies (Vanilla ES Modules, CSS variables, and HTML5 semantic markup).
    • Privacy Centric: No database cookies, tracking systems, or background tracking APIs. Your financial history lives fully within your device data bounds.
    • Accessible Controls: Native compliance with assistive tech loops, ARIA live regions for budget updates, and strict focus state capture.
